   - Any other non-empty string: set **`:unknown`** **or** omit override — pick **one** behavior and document it in a **one-line** comment next to the clause; recommended: treat invalid as **`:unknown`** via explicit **`config :kiln, :operator_runtime_mode, :unknown`** so **`OperatorRuntime.mode/0`** matches runtime intent.
