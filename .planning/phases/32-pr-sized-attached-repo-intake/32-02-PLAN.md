| Boundary | Description |

| promoted request -> run creation | The launcher converts one promoted bounded request into a durable run row that later stages rely on. |
| runs context -> run director | A bad launch seam could create an untracked run or bypass existing start guards. |
| run identity -> later attached phases | Missing or JSON-only links would break continuity and reviewability in later phases. |
