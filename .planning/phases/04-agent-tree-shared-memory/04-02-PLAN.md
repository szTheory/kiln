Phase 4 minimal handoff contract locked by this plan:

```text
Run boot -> Mayor seeds one initial planner-tagged work unit when the run has none
Planner claims :planner units only; may create successor units tagged :coder, :reviewer, :uiux, :tester, :qa_verifier, or :mayor
Coder claims :coder units only
Reviewer claims :reviewer units only
UIUX claims :uiux units only
Tester claims :tester units only
QAVerifier claims :qa_verifier units only
Mayor claims :mayor units only

Unblock means: dependency edges causing the block are cleared, `blockers_open_count` returns to 0, and an `:unblocked` event is appended
Handoff completion means: the current unit records `:completed` then `:closed`, successor units record `:created`, and PubSub fires only after commit
```
