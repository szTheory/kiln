Create **`lib/kiln_web/live/run_compare_live.ex`**:

1. **`use KilnWeb, :live_view`** (same as other lives).
2. **`@impl true def mount(_params, _session, socket)`** → `{:ok, assign(socket, :page_title, "Compare runs")}` (or exact string from UI-SPEC: **"Compare runs"**).
3. **`handle_params(params, _uri, socket)`**:
   - Read **`params["baseline"]`** and **`params["candidate"]`**.
   - **If either string is present and `Ecto.UUID.cast/1` returns `:error` for that value:** return **`{:noreply, socket |> put_flash(:error, "Invalid run id") |> push_navigate(to: ~p"/")}`** (mirror detail copy **"Invalid run id"`**).

4. **`render/1`**: wrap in **`<Layouts.app flash={@flash} current_scope={@current_scope} factory_summary={@factory_summary}>`** (match **`RunBoardLive`** / **`RunDetailLive`** assigns — **same** arity as siblings).
5. Inner wrapper **`<div id="run-compare" …>`** with minimal placeholder headings **Baseline** / **Candidate** (can be empty columns).
6. **`@impl true def handle_event("noop", _, socket), do: {:noreply, socket}`** stub optional.

**Do not** implement union spine, **`Runs.Compare`**, or PubSub in this plan.
