Minimal role handoff contract this plan must implement:

```text
Mayor boot responsibility:
- On subtree boot, Mayor calls `Kiln.WorkUnits.seed_initial_planner_unit/1`
- If the run already has any open/in_progress work unit, Mayor does nothing

Claim matrix:
- Mayor -> :mayor only
- Planner -> :planner only
- Coder -> :coder only
- Tester -> :tester only
- Reviewer -> :reviewer only
- UIUX -> :uiux only
- QAVerifier -> :qa_verifier only

Minimum event kinds already locked by 04-02:
- :created, :claimed, :blocked, :unblocked, :completed, :closed

Completion/handoff:
- A role never mutates successor rows directly through Repo
- It calls `Kiln.WorkUnits.complete_and_handoff/3`
- Current unit gets :completed then :closed
- Successor units are inserted with :created in the same transaction
```
