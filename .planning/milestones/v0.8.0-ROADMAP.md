# Milestone v0.8.0 — Roadmap

## Goal
Transition Kiln from a local-only tool into a remote-capable factory that can autonomously merge, release, and verify deployments.

## Phases

### Phase 36: Remote Access & Operator Auth
Secure the factory for remote use.
- **36-01**: Implement single-operator Auth (Password/Passkey) for the Phoenix dashboard.
- **36-02**: Add a `remote` Docker Compose profile with Tailscale/Cloudflare sidecar.
- **36-03**: Verify remote connectivity and Auth gate posture.

### Phase 37: Autonomous Versioning & Release
Automate the software release lifecycle.
- **37-01**: Build a Conventional Commits parser and auto-versioning engine.
- **37-02**: Implement GitHub Release integration for binaries and multi-arch artifacts.
- **37-03**: Implement Registry Publish integration (Hex/npm) for libraries.

### Phase 38: SaaS Delivery & SRE Smoke
Close the loop with live deployments.
- **38-01**: Implement SaaS deployment drivers (Fly.io/Cloudflare) using host CLIs.
- **38-02**: Build the post-deploy Health Check worker and escalation logic.
- **38-03**: Surface live URLs and SRE status tiles on the Run Detail view.

### Phase 39: Milestone Proof & Production Readiness
Final validation of the closed-loop factory.
- **39-01**: Execute a full "closed loop" run: Commit → Merge → Release → Deploy → Verify.
- **39-02**: Cleanup planning debt and finalize v0.8.0 documentation.
