Brand: Kiln

Essence:
Kiln is a software dark factory. The brand stands for durable systems, calm operators, visible work, disciplined craft, and controlled heat. Kiln turns uncertain software work into durable, inspectable, repeatable systems.

Voice:
Precise, calm, grounded, competent, restrained.
No hype, no slang, no fake excitement, no “AI magic.”
Short sentences. Concrete nouns. Active verbs.
Always say what happened, what happens next, and what the user can do.

Visual style:
Dark workshop energy. Coal, clay, ash, bone, ember.
Premium operator UI. Tactile but minimal.
Use warm accents sparingly.
No cyberpunk, no sparks, no flames, no robots, no factory clip art.

Typography:
Primary sans: Inter
Mono: IBM Plex Mono
Optional display: Geist for marketing only

UI rules:
Rectangles first. Slightly softened corners.
Borders over heavy shadows.
Clear state hierarchy.
Every component has loading, empty, success, warning, error, focus, disabled states.
Operator clarity beats decorative flourish.

Imagery:
Fired clay, graphite, ceramic, steel, paper, measured heat, schematics, timelines, artifacts, dashboards, inspection marks.

Avoid:
Crypto vibes, cyberpunk neon, fantasy forge imagery, AI brains, cute mascots, loud gradients.

Microcopy:
Use direct stateful language like:
Start run
Resume run
Verify changes
Promote build
View trace
Build verified
Verification failed
Retry step
Waiting on upstream
Manual review required
